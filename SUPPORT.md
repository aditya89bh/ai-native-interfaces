# Support

Thanks for using `ai-native-interfaces`. Here is how to get help.

## Documentation first

Most questions are answered by the docs:

- [README](README.md) — overview, installation, and quick start.
- [Storybook](https://aditya89bh.github.io/ai-native-interfaces/) — every component with live examples (published from `main`).
- [API reference](docs/api-reference.md) — all public exports and props.
- [Theming](docs/theming.md), [Accessibility](docs/accessibility.md), and the [documentation index](README.md#documentation).

## Questions and ideas

- Use [GitHub Discussions](https://github.com/aditya89bh/ai-native-interfaces/discussions) for how-to questions, ideas, and show-and-tell.

## Bugs and feature requests

- Open an [issue](https://github.com/aditya89bh/ai-native-interfaces/issues/new/choose) using the appropriate template. Include a minimal reproduction for bugs.

## Security

- Do not report vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md).

## Response expectations

This is an open-source project maintained on a best-effort basis. Issues and discussions are triaged as time allows; clear, reproducible reports get resolved fastest. Contributions are welcome — see [CONTRIBUTING.md](CONTRIBUTING.md).
